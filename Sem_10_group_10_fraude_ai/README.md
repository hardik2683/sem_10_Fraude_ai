# FraudGuard AI – Fraud Detection Web Application

A hybrid ML-powered fraud detection system built with Flask and scikit-learn.
No labelled training data required. CPU-only. Runs locally in minutes.

## Architecture

```
Isolation Forest (40%) ──┐
Local Outlier Factor (20%) ──┼── Weighted Score (0-100) ──► Verdict
Rules Engine (40%) ──────┘
```

## Folder Structure

```
fraud_detection/
├── app.py                      # Flask application (routes, scoring)
├── create_model.py             # One-time model initialisation script
├── requirements.txt
├── model/
│   └── fraud_model.pkl         # Generated by create_model.py
├── templates/
│   └── index.html              # Frontend UI
├── static/
│   └── style.css               # Styles
└── utils/
    ├── __init__.py
    └── feature_engineering.py  # Feature extraction + rules engine
```

## Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Build the model (run once)
python create_model.py

# 3. Start the server
python app.py
```

Open http://127.0.0.1:5000 in your browser.

> **Note:** `create_model.py` runs automatically on first launch if
> `fraud_model.pkl` is missing — so you can skip step 2 if you prefer.

## Input Fields

| Field                   | Description                            |
|-------------------------|----------------------------------------|
| Amount (USD)            | Transaction dollar value               |
| Transaction Type        | online_purchase / atm_withdrawal / …   |
| Location Category       | local / domestic / international / …   |
| Transaction Hour        | 0–23 (hour of day)                     |
| Device Type             | mobile / desktop / tablet / atm        |
| Recent Transaction Count| How many transactions in recent window |
| Account Age (days)      | Days since account was created         |
| Failed Auth Attempts    | Recent failed logins                   |
| New Device              | Checkbox — unrecognised device         |

## Verdict Thresholds

| Score     | Verdict    | Risk Level |
|-----------|------------|------------|
| 60 – 100  | FRAUDULENT | HIGH       |
| 35 – 59   | SUSPICIOUS | MEDIUM     |
| 0  – 34   | LEGITIMATE | LOW        |
